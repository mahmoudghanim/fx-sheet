# 🚀 How to Run FX Sheet Dashboard on Windows

## Quick Start (Easiest Method)

### Method 1: Double-Click to Start
1. **Double-click** `Start Dashboard.bat`
2. The dashboard will **automatically open** in your browser
3. **Keep the window open** while using the dashboard
4. To stop: Close the window or press `Ctrl+C`

### Method 2: PowerShell (Better Visual)
1. **Right-click** `Start Dashboard.ps1`
2. Select **"Run with PowerShell"**
3. The dashboard will open automatically
4. Keep the window open while using

---

## Accessing the Dashboard

Once started, the dashboard is available at:
- **URL**: http://localhost:8001
- The browser opens automatically when you start the server

---

## Updating Stock Data

To fetch the latest stock prices:

### Option 1: Double-Click
1. Double-click `update_data.bat`
2. Wait for the update to complete (~5-10 minutes)
3. Refresh your browser (Ctrl + Shift + R)

### Option 2: Command Line
```
python fetch_yahoo_data.py
```

## Troubleshooting

### "Python is not installed" Error
1. Download Python from: https://www.python.org/downloads/
2. During installation, **check "Add Python to PATH"**
3. Restart your computer
4. Try running `Start Dashboard.bat` again

### Dashboard Won't Open
1. Make sure `server.py` is running (window should be open)
2. Manually open: http://localhost:8001
3. Check if port 8001 is already in use

### Data Not Showing
1. Run `update_data.bat` to fetch fresh data
2. Wait for completion (shows progress)
3. Refresh browser with Ctrl + Shift + R
