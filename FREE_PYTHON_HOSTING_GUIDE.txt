================================================================================
DEPLOY TO PYTHONANYWHERE - STEP BY STEP
================================================================================

STEP 1: Sign Up
---------------
1. Go to: https://www.pythonanywhere.com
2. Click "Create a Beginner account" (FREE)
3. Choose username: (this will be in your URL)
4. Enter email and password
5. Verify email

STEP 2: Upload Files
--------------------
1. Login to PythonAnywhere
2. Go to "Files" tab
3. Click "Upload a file"
4. Upload these files:
   ✓ index.html
   ✓ stocks-data.json
   ✓ flask_app.py
   ✓ requirements.txt

STEP 3: Create Web App
----------------------
1. Go to "Web" tab
2. Click "Add a new web app"
3. Choose "Manual configuration"
4. Select Python 3.10 (or latest available)
5. Click "Next"

STEP 4: Configure WSGI File
----------------------------
1. Click on "WSGI configuration file" link
2. Delete ALL existing code
3. Add this code:

```python
import sys
path = '/home/yourusername/fx-sheet'
if path not in sys.path:
    sys.path.append(path)

from flask_app import app as application
```

4. Replace "yourusername" with your actual username
5. Save the file (Ctrl + S)

STEP 5: Set Up Virtual Environment
-----------------------------------
1. Go to "Web" tab
2. In "Virtualenv" section, enter:
   /home/yourusername/.virtualenvs/fx-sheet

3. Click "Enter" to create it

STEP 6: Install Flask
---------------------
1. Open Bash console (from Consoles tab)
2. Run these commands:

```bash
# Create virtual environment
mkvirtualenv --python=/usr/bin/python3.10 fx-sheet

# Install Flask
pip install flask==3.0.0

# Or install from requirements.txt
cd /home/yourusername
pip install -r requirements.txt
```

STEP 7: Set Static Files
-------------------------
1. Go to "Web" tab
2. Scroll to "Static files" section
3. Add these mappings:

URL: /
Path: /home/yourusername/

OR better:
URL: /static
Path: /home/yourusername/

4. Click "Add" after each

STEP 8: Reload Web App
----------------------
1. Go to "Web" tab
2. Click the green "Reload" button
3. Wait for confirmation

STEP 9: Test Your Dashboard
----------------------------
Visit: https://yourusername.pythonanywhere.com/

You should see:
✓ Dashboard loads
✓ All 463 stocks display
✓ Filters work
✓ Sorting works
✓ Yahoo Finance links work

STEP 10: Enable Stock Management API (Optional)
-------------------------------------------------
If you want add/remove stocks from dashboard:

1. Edit flask_app.py to include API endpoints
2. Already included in your flask_app.py!
3. Just make sure it's uploaded

================================================================================
TROUBLESHOOTING PYTHONANYWHERE
================================================================================

Problem: "Application Error"
Fix:
  1. Check error log (Web tab → Error log)
  2. Verify WSGI file is correct
  3. Make sure Flask is installed
  4. Check file paths in WSGI

Problem: "ModuleNotFoundError: flask"
Fix:
  1. Open Bash console
  2. Activate virtualenv: workon fx-sheet
  3. Install Flask: pip install flask
  4. Reload web app

Problem: Dashboard loads but no data
Fix:
  1. Check stocks-data.json is uploaded
  2. Verify file location
  3. Check browser console (F12)
  4. Try: yourusername.pythonanywhere.com/stocks-data.json

Problem: Cannot fetch from Yahoo Finance
Fix:
  1. Yahoo Finance IS whitelisted
  2. If blocked, check whitelist:
     https://www.pythonanywhere.com/whitelist/
  3. Use pre-fetched data instead

Problem: Stock management API doesn't save
Fix:
  1. Check file permissions
  2. Make sure flask_app.py has save endpoint
  3. Check error log for details
  4. Verify virtual environment is set
