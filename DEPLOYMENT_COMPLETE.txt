================================================================================
🐛 TROUBLESHOOTING
================================================================================

Problem: Dashboard not loading
Solution: Check GitHub Pages settings are correct
         Verify files are in repository

Problem: Stock data not updating
Solution: Check Actions tab for errors
         Verify yfinance is working
         Check workflow file syntax

Problem: Update failed in Actions
Solution: Click on failed run to see error log
         Common issues: Yahoo Finance rate limits
         Retry manually if needed

Problem: Changes not appearing
Solution: GitHub Pages caches for 1-2 minutes
         Hard refresh browser: Ctrl+Shift+R
         Check Actions completed successfully
