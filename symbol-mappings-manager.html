🔗 Symbol Mappings Manager

Fix incorrect Yahoo Finance symbols for your stocks

How to Use

Add custom mappings to fix incorrect stock symbols. The system will use these mappings instead of the automatic conversion when fetching data from Yahoo Finance.

Example: If "ABBV.us" is fetching wrong data, map it to the correct Yahoo symbol like "ABBV"

⚙️ GitHub Token Setup (One-Time)

To save mappings directly to GitHub, you need a Personal Access Token.

Steps:
1. Go to GitHub Token Settings
2. Click "Generate new token (classic)"
3. Give it a name (e.g., "FX Sheet Manager")
4. Select scope: repo (full control of private repositories)
5. Click "Generate token" and copy it
6. Click the "Set Token" button below and paste your token
